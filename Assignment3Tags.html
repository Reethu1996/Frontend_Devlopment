There are two types of Tags in HTML

  1. Paired Tags
  2. UnPaires Tags

Paired Tags:

A Tag which has both opening and closing 

Examples: Para tag, header tag, body tags etc..,.

Unpaired Tag:

A Tag which does not have closing 

Examples: break, horizontal line, img tags etc...,
